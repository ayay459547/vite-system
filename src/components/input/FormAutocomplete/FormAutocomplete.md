### Attributes ( props )

| 屬姓名          | 說明                    | 類型    | 預設值    |
| --------------- | ---------------------- | ------- | --------- |
| errorMessage    | 錯誤訊息 驗證用         | string  | ''        |
| modelValue      | v-model                | string  | number | undefined |
| placeholder     | 輸入框佔位文字          | string  | undefined |
| clearable       | 是否顯示清除按鈕        | boolean  | false   |
| disabled        | 是否禁用                | boolean | false   |
| valueKey        | 選項取值 用的key        | string   | 'value' |
