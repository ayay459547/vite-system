### Attributes ( props )
| 屬姓名     | 說明         | 类型    | 默認值            |
| --------- | ------------ | ------- | ---------------- |
| icon      | 圖示          | Array  | []               |
| type      | 圖示類型      | enum    | 'fas'            |
| name      | 圖示名稱      | string  | 'circle-question'|
