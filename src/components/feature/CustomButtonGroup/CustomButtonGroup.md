## CustomButtonGroup

### 使用範例

```vue
<script setup lang="ts">
import { CustomButton, CustomButtonGroup } from '@/components'

</script>

<template>
  <CustomButtonGroup>
    <CustomButton label="Button1" />
    <CustomButton label="Button2" />
  </CustomButtonGroup>
</template>
```
